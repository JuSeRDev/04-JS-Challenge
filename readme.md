# Base Apparel Coming Soon Page

Una página de "Coming Soon" para la tienda de moda Base Apparel. El diseño es limpio, minimalista y responsivo, con un formulario de suscripción que incluye validación de correo electrónico.

## Características
- Diseño responsivo que se adapta a diferentes tamaños de pantalla.
- Formulario de suscripción con validación de correo electrónico.
- Estilo moderno y minimalista con uso de degradados y tipografías atractivas.
- Mensajes de error y retroalimentación visual para el usuario.
- Imágenes específicas para vistas de escritorio y móviles.

## Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox y media queries)
- JavaScript (validación de formularios)
- Google Fonts (Josefin Sans)
- Imágenes responsivas

## Explicación del código

Este código crea un componente de "Coming Soon". Se seleccionan elementos del DOM como el formulario de suscripción, los mensajes de error y el botón de envío. Cuando se hace clic en el botón de envío, se valida el correo electrónico ingresado. Si el formato es incorrecto, se muestra un mensaje de error. Además, se utilizan imágenes responsivas para ofrecer una experiencia de usuario óptima en diferentes dispositivos.
